Nivel principiante: 1. Diseño básico del sitio web: Cree un archivo HTML con la estructura básica de una página web. Incluya un encabezado, una barra de navegación, una sección de contenido principal y un pie de página. Utilice las clases Bootstrap para formatear los elementos de la página. 2. Tarjetas receptivas: Cree un contenedor con varias tarjetas Bootstrap. Utilice la clase col-sm-* para que las tarjetas sean receptivas y se adapten al tamaño de la pantalla. 3. Cuadrícula de imágenes: Cree una cuadrícula de imágenes utilizando la clase row y col-md-*. Asigne a cada imagen una clase de tarjeta Bootstrap. 4. Mensajes de alerta: Cree diferentes tipos de alertas Bootstrap (éxito, advertencia, peligro, etc.). Utilice la clase alert-* para mostrar el tipo de alerta. 5. Pestañas de navegación: Cree una barra de navegación con pestañas utilizando la clase nav-tabs. Asigne a cada pestaña una clase nav-item.
